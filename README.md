-프로젝트 개요
본 프로젝트는 파이썬의 기본 문법인 조건문, 반복문, 함수, 자료구조를 활용하여 확장형 무인 주문 시스템을 구현하는 것을 목표로 한다.
사용자가 메뉴를 선택하고 수량을 입력하여 여러 개의 상품을 주문할 수 있도록 장바구니 기능을 포함하였다.
또한 잘못된 입력에 대한 오류 처리를 통해 프로그램의 안정성을 높였다.
이를 통해 실제 무인 주문 시스템의 동작 원리를 이해하고, 구조적인 프로그램 설계 능력을 기르고자 한다.

-주요 기능
메뉴 목록 출력 기능
딕셔너리를 사용하여 메뉴 이름과 가격을 저장하고, 이를 화면에 출력하여 사용자가 주문할 수 있도록 한다.

메뉴 선택 및 수량 입력 기능
사용자가 원하는 메뉴 번호와 수량을 입력할 수 있으며, 여러 번 반복하여 주문이 가능하도록 구현하였다.

장바구니 기능
사용자가 선택한 메뉴와 수량을 리스트에 저장하여 여러 개의 주문을 관리할 수 있도록 하였다.

주문 내역 및 총 결제 금액 출력 기능
장바구니에 담긴 주문 내역을 출력하고, 각 메뉴의 가격과 수량을 계산하여 총 결제 금액을 표시한다.

입력 오류 처리 기능
존재하지 않는 메뉴 번호나 잘못된 수량 입력 시 오류 메시지를 출력하여 프로그램이 정상적으로 동작하도록 한다.

-실행 방법
프로그램을 실행하면 메뉴가 출력된다.
메뉴 번호와 수량을 입력하여 주문하며, 0을 입력하면 주문이 종료된다.
이후 주문 내역과 총 결제 금액이 화면에 출력된다.
